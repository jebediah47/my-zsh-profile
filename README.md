# my-zsh-profile
My .zshrc config

To install clone the repo:
```bash
git clone https://github.com/jebediah47/my-zsh-profile
```

Then cd into the repo and make the installation script executable
```bash
cd my-zsh-profile
chmod +x ./install.sh
```

After doing that just run the installation script
```bash
./install.sh
```
